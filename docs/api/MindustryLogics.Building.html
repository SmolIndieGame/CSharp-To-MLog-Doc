Class Building

An instance of a building.
This class contains all information about a building.
Do stuff with the building by accessing its member, for example,
b.Shoot(10, 10, true) will make building b shoot at (10, 10).

Radar(RadarFilter, SortMethod, Boolean)

Locate units around this building.
Unable to detect units outside the Range of this building.

Example:
turret1.Radar(Filter.Radar().Enemy, SortMethod.Distance, true)
This will return a enemy that is in range and is closest to turret1.

Declaration
public abstract Unit Radar(RadarFilter target, SortMethod sortMethod, bool firstOrLast)
Parameters
Type Name Description
RadarFilter target

The target filter.
Use Radar() to create the filter than add constraints into it. (maximum three constraints)

Example:
To target enemy ground unit: Filter.Radar().Enemy.Ground
To target anything: Filter.Radar() or null
SortMethod sortMethod

How to sort the results if there are multiple units that are in range and pass the filter.

Example:
To sort by health: SortMethod.Health
System.Boolean firstOrLast

Returns the first unit in the results sorted by sortMethod if true, returns the last unit otherwise.
