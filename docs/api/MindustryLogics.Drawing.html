Class Drawing

This class contains all drawing operations.

Coordinate inside displays:

  • The size of a Logic Display is 80x80.
  • (x = 0, y = 0) is the bottom left corner.
  • (x = 80, y = 80) is the upper right corner.
  • The size of a Large Logic Display is 176x176.
  • (x = 0, y = 0) is the bottom left corner.
  • (x = 176, y = 176) is the upper right corner.

DrawImage(Double, Double, Object, Double, Double)

Draw an icon of a content of type image centered at (centerX, centerY) with size size.
The image will be rotated counter-clockwise rotation degrees.
image can only be either a BuildingType, UnitType, ItemType, LiquidType.
Other types will cause this method to draw a white square with red text "ohno" on it.
The size of the icon is the same as the diameter of the icon.

System.Double centerX
System.Double centerY
System.Object image
System.Double size

Remarks

This method uses the pen's color.
Does not do anything until DrawFlush(Building) or DrawFlush(Int32) is used.
