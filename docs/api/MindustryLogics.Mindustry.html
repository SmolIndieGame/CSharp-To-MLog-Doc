BindingUnit

The current bound Unit. It changes when UnitBind(UnitType) is called,
or set to null when Unbind() is called.

Logics can only manipulate the current bound unit (control, radar...).
However, to sensor (get information about) a unit,
you can reference the property inside that Unit.
Example: u.Flag will returns the flag of the unit u,
but u.Move(...) is not allowed.

If you want to move the current bound unit, use Move(Double, Double).

UnitRadar(RadarFilter, SortMethod, Boolean)

Locate units around the current bound unit.
Unable to detect units outside the Range of the current bound unit.

For locating units using buildings, use Radar(RadarFilter, SortMethod, Boolean) instead.

Example:
Mindustry.UnitRadar(Filter.Radar().Enemy, SortMethod.Distance, true)
This will return a enemy that is in range and is closest to the current bound unit (BindingUnit).

Declaration
public static Unit UnitRadar(RadarFilter target, SortMethod sortMethod, bool firstOrLast)
Parameters
Type Name Description
RadarFilter target

The target filter.
Use Radar() to create the filter than add constraints into it. (maximum three constraints)

Ex:
To target enemy ground unit: Filter.Radar().Enemy.Ground
To target anything: Filter.Radar() or null
SortMethod sortMethod

How to sort the results if there are multiple units that are in range and pass the filter.

Ex: Sort by health: SortMethod.Health
System.Boolean firstOrLast

Returns the first unit in the results sorted by sortMethod if true, returns the last unit otherwise.
Returns
Type Description
Unit
Remarks

This method uses the current bound unit. Bind a unit with UnitBind(UnitType) and make sure that BindingUnit is not null.
