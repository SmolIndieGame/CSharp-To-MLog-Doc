Build(Double, Double, BuildingType, Double, Building)

The x coordinate to build at.
System.Double y

The y coordinate to build at.
BuildingType buildingType

The type of the building to build.
System.Double rotation

The roataion of the building to build.
For buildings that have directions:

  • 0 faces right
  • 1 faces up
  • 2 faces left
  • 3 faces down
For turrets and units:

  • 0 faces right
  • 90 faces up
  • 180 faces left
  • 270 faces down
Building config

The building to copy from.
If the type of the building config is same as buildingType,
then the building to build will have the same properties as config.
It is similar to a player pressing "f" on the building config and placing it on (x, y).

GetBlock(Double, Double, out BuildingType, out Building)

Get the block information at location (x, y).
The current bound unit must be in range of the location.

Example:
UnitControl.GetBlock(x, y, out t, out b)
This will get the block information at (x, y) and store the type of the block in t, the building in b.
(You need to define t and b prior to calling this method)

Declaration
public static void GetBlock(double x, double y, out BuildingType type, out Building building)
Parameters
Type Name Description
System.Double x

The x coordinate to get information from.
System.Double y

The y coordinate to get information from.
BuildingType type

The type of the block at location (x, y).
Solid non-buildings (environment walls) will have type Solid.
Building building

The building at location (x, y).
It can be null if there is no block or the block is an environment wall.
Remarks

Unbind()

Unbind the current bound unit, making this logic no longer able to manipulate this unit.
Normally, an unit controlled by a processor will unbind after some time if the processor is no longer controlling it.
Calling this function will make the unit switch back to using the default unit AI instantly.

Not available on V6.

Declaration
public static void Unbind()
Remarks

This method uses the current bound unit. Bind a unit with UnitBind(UnitType) and make sure that BindingUnit is not null.
