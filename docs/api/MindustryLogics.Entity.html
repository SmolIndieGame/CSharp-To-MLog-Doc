Rotation

The roataion of the entity.
For buildings that have directions:

  • 0 faces right
  • 1 faces up
  • 2 faces left
  • 3 faces down
For turrets and units:

  • 0 faces right
  • 90 faces up
  • 180 faces left
  • 270 faces down
